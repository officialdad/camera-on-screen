# M5 Bundler — verification (RTX 3090)

Date: 2026-06-21 (headless gate + measurement) / 2026-06-22 (human visual gate)
Branch: `feat/m5-bundler`
Spec/plan: `docs/superpowers/specs/2026-06-21-camera-on-screen-m5-bundler-design.md`, `docs/superpowers/plans/2026-06-21-camera-on-screen-m5-bundler.md`

## What was verified

`scripts/bundle-maxine.ps1` produces a pruned `<output>\maxine\` from the co-versioned
runtimes (VFX 0.7.6 at `…\VideoFX-0.7.6`, AR 0.8.7 at `%ProgramFiles%\NVIDIA Corporation\NVIDIA AR SDK`),
and both Maxine effects load + run from that pruned bundle with **no `COS_*` env vars set**.

## Size — pruned vs verbatim

| Bundle | Size |
|--------|------|
| `maxine.full` (verbatim two-tree copy, pre-prune) | 2.024 GB (2,173,067,215 B) |
| `maxine` (pruned, bundler output) | **1.277 GB (1,371,215,295 B)** |
| Saved | ~0.75 GB (**~37%**) |

Per-section: root DLLs 17 files / 1,127 MB · `models\vfx` 8 / 112 MB · `models\ar` 7 / 66 MB · license 2 files.

Closure (from `native/shim/smoke/trace_closure.cpp`): 13 shared DLLs sourced from the VFX
runtime; **2 AR-only** DLLs (`cufft64_11.dll`, `nppif64_12.dll`) sourced from the AR runtime
(gaze pulls FFT + the `nppif` primitive; VFX 0.7.6 ships neither — same CUDA 12.1 generation,
co-version-safe). Dropped vs verbatim: `cublas64_12`, `nppicc64_12`, `nvinfer_plugin_10`,
`nvrtc*`, `nvvpi2`, `cufft64_10`, `cufftw64_*`, and non-compute-86 model variants.

## Verify gate (automatable)

`trace_closure.exe` re-run against the produced pruned `maxine\`, all `COS_*` unset, CWD in the output dir:

```
# green-screen Start=1  eye-contact Start=1
# 17 modules loaded from maxine\:
cublaslt64_12 cudart64_12 cufft64_11 libcrypto-3-x64 nppc64_12 nppial64_12 nppidei64_12
nppif64_12 nppig64_12 nppim64_12 nppist64_12 nppitc64_12 nvarpose nvcvimage nvinfer_10
nvonnxparser_10 nvvideoeffects
gate exit=0
```

All 17 loaded from the bundle — no env-var fallback. Engines deserialize on the RTX 3090
(co-version holds).

## Human visual gate (definitive — `2026-06-22`)

App launched from the publish dir with `COS_*` unset (app-relative `maxine\` tier). Confirmed:
both capability probes pass, both toggles enable, **green screen works, eye contact works, and
both together** on the RTX 3090.

## Result

**PASS.** The bundler produces a minimal, self-contained, co-versioned runtime that both
effects load and run from with zero environment configuration. Remaining ship-time work
(installer, multi-GPU models, full license review, release.yml) is tracked as GitHub issues.
